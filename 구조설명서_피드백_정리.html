2. Timestamp / PLC 관련

필수 수정timestamp는 PLC 기준으로 필수 수정
PLC 값PLC 값 자체 수정 필요 ("plc 값 고쳐야함" 메모)
제약사항런타임에서 PLC 관련 값은 설비 상태 값이 변할 때 이벤트성으로만 처리, 함수를 통해서 반영

4. 모델 폴더/경로 규칙

결정모델 폴더를 DLL과 같은 경로에 두지 않음
결정별도 모델 폴더를 두고 이름으로 구별
추후 결정모델 네이밍 룰은 추후 확정

Input 구성Input에 JSON 파일 포함
