LGES IVS algorithm DLL(Input/Output 명세서)

📥 Input 명세 (CNS → LGES DLL)

실제 코드 구조:

struct DetectInputDto {
    std::shared_ptr<Frame> frame;  // 프레임 (이미지 + 메타정보)
    Plc_Info plcInfo;              // PLC 정보
};

struct Frame {
    uint64_t index;                                  // 프레임 번호
    cv::Mat data;                                    // 이미지 데이터
    std::chrono::system_clock::time_point timestamp; // 수신 시각
    int ref_count;                                   // 참조 카운트
};

struct Plc_Info {
    int equip_status;  // 설비 상태
};

Input 통합표

구조 항목명 타입 필수 제공 주기 제공 방식 설명 예시/제약
Frame index uint64_t ✓ 매 프레임 직접 제공 프레임 번호 (순차 증가) 0, 1, 2, 3, ...

timestamp chrono::time_point ✓ 매 프레임 직접 제공 프레임 수신 시각 Frame 생성자에서 자동 설정
Plc_Info equip_status int ✓ 이벤트성 직접 제공 설비 상태 상태 코드 값
설정 파일 config json file ✓ 이벤트성 파일 경로 전체 시스템 설정 아래 상세 표 참조

📤 Output 명세 (LGES DLL → CNS)

실제 코드 구조:

// 콜백: Fault/Anomaly 검출 시
class IFaultListener {
    virtual bool OnFaultDetected(
        const DetectInputDto& input_dto,
        const DetectOutputDto& dod
    ) = 0;
};

struct DetectOutputDto {
    std::shared_ptr<Frame> frame;      // 원본 프레임
    cv::Mat visualized_img;            // 시각화 이미지
    unsigned int takt_time;            // 처리 시간 (ms)
    std::vector<ResultItem> ab_results;// 구조적 결함 목록
    std::vector<ResultItem> q_results; // 품질 이상 목록
    ErrorCodeEnum error_code;          // 에러 코드
    std::string error_msg;             // 에러 메시지
};

struct ResultItem {
    DefectTypeEnum defect_type;  // 결함 타입
    float score;                 // 신뢰도
    Box box;                     // 바운딩 박스
};

struct Box {
    unsigned int x1, y1;  // 좌측 상단
    unsigned int x2, y2;  // 우측 하단
};

Output 통합표

콜백 이름 파라미터
구조		 항목명 타입 항상
존재		 설명 예시/제약
OnFaultDetected - input_dto DetectInputDto ✓ 원본 Input 전체 frame + plcInfo 전체 구조
(결함/이상 검출 시) - input_dto.frame shared_ptr<Frame> ✓ 원본 프레임 index, data, timestamp, ref_count
- input_dto.plcInfo Plc_Info ✓ 원본 PLC 정보 equip_status
dod. frame shared_ptr<Frame> ✓ 원본 프레임 포인터 input_dto.frame과 동일
dod. visualized_img cv::Mat △ 시각화 이미지 비어있을 수 있음 (empty Mat)
dod. takt_time unsigned int ✓ 처리 시간 (ms) 0~, 기본값 0
dod. ab_results vector<ResultItem> ✓ 구조적 결함 목록 0개 이상 (빈 배열 가능)
dod.ab_results[i]. defect_type DefectTypeEnum ✓ 결함 타입 모델 의존
dod.ab_results[i]. score float ✓ 신뢰도 0.0 ~ 1.0
dod.ab_results[i]. box.x1 unsigned int ✓ 좌측 상단 X 픽셀 좌표
dod.ab_results[i]. box.y1 unsigned int ✓ 좌측 상단 Y 픽셀 좌표
dod.ab_results[i]. box.x2 unsigned int ✓ 우측 하단 X 픽셀 좌표
dod.ab_results[i]. box.y2 unsigned int ✓ 우측 하단 Y 픽셀 좌표
dod. q_results vector<ResultItem> ✓ 품질 이상 목록 0개 이상 (빈 배열 가능)
dod.q_results[i]. defect_type DefectTypeEnum ✓ 결함 타입 모델 의존
dod.q_results[i]. score float ✓ 신뢰도 0.0 ~ 1.0
dod.q_results[i]. box.x1 unsigned int ✓ 좌측 상단 X 픽셀 좌표
dod.q_results[i]. box.y1 unsigned int ✓ 좌측 상단 Y 픽셀 좌표
dod.q_results[i]. box.x2 unsigned int ✓ 우측 하단 X 픽셀 좌표
dod.q_results[i]. box.y2 unsigned int ✓ 우측 하단 Y 픽셀 좌표
dod. error_code ErrorCodeEnum ✓ 에러 코드 0=NO_ERROR_ (기본값)
dod. error_msg string ✓ 에러 메시지 비어있을 수 있음
- [반환값] bool ✓ 처리 계속 여부 true=계속, false=중단
